# json-server-todos

# Installation

- npm install
- cd server
- npm start

Now opens:

- http://localhost:9001

You now have a full REST API. Test with POSTMAN or any other REST Client):

Retrieve all (GET):

```bash
GET http://localhost:9001/todo_list
```

Post a todo (POST):
http://localhost:9001/todo_list
